Import Data
Import Chart of Accounts, Journal Entries, Parties and Project Budgets
📂
Chart of Accounts
Account codes, names, classes & hierarchy
Import your full account structure from Excel or CSV. Existing codes are updated; new codes are added. Supports parent/sub-account hierarchy.
Required columns: Code, Account Name, Class (ASSET/LIABILITY/CAPITAL/INCOME/EXPENSE), CF Category, Parent Code
📒
Journal Entries (GL)
Transactions, vouchers & accounting lines
Import historical transactions. Rows with the same Voucher No are grouped into one entry. Duplicate voucher numbers are automatically skipped.
Required columns: Date, Voucher No, Account Code, Debit, Credit | Optional: Voucher Type, Narration, Project Code, Party Name
👥
Parties / Contacts
Vendors, employees, customers & consultants
Import your full contact directory. Type determines the auto-code prefix (EMP, VND, CST, CON). Duplicate names are skipped.
Required columns: Name, Type (Employee/Vendor/Customer/Consultant/Other) | Optional: Code, Contact
📊
Project Budgets
Budget lines by project & category
Import budget allocations per project by category. Used in the Budget vs. Actual report. Existing project budgets are overwritten with imported values.
Required columns: Project Code, Total Budget | Optional: Personnel, Travel, Equipment, Consultancy, Overhead, Other, Budget From, Budget To
💡 Import Tips:   Always download the template first to ensure correct column headers.  ·  Import CoA before GL (account codes must exist).  ·  Files must be .csv or .xlsx format.  ·  Take a backup before large imports.
